# Задание 1

## Задача 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для

последовательности

### Алгоритм решения

1\. Подключить пространство имён System для работы с консолью.

2\. Определить функцию truncateNumber, которая принимает вещественное число и возвращает целое, отбрасывая дробную часть.

3\. Определить рекурсивную функцию для проверки корректного ввода вещественного числа от пользователя.

4\. Внутри функции считать строку с помощью Console.ReadLine().

5\. Проверить корректность ввода с помощью Double.TryParse.

6\. Если ввод корректный, вернуть введённое число.

7\. Если ввод некорректный, вывести сообщение об ошибке и повторно вызвать функцию для ввода.

8\. Объявить точку входа программы [<EntryPoint>] let main argv =.

9\. Вывести пользователю сообщение "Введите количество элементов последовательности:".

10\. Считать с консоли количество элементов и преобразовать его в int, сохранить в count.

11\. Вывести пользователю сообщение "Введите вещественные числа:".

12\. Создать последовательность вещественных чисел floatSequence, используя seq { for _ in 1 .. count -> readFloat() }.

13\. Преобразовать каждый элемент последовательности в целое с помощью Seq.map truncateNumber, сохранить результат в intSequence.

14\. Вывести сообщение "Последовательность без дробной части:".

15\. Пройти по всем элементам intSequence с помощью Seq.iter (printf "%d ") и вывести их на экран.

16\. Вывести пустую строку printfn "" для переноса строки после последовательности.

17\. Завершить работу программы, возвращая 0.

# Задание 2

## Задача 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для

последовательности

### Алгоритм решения

1\. Подключить пространство имён System для работы с консолью.

2\. Определить рекурсивную функцию для проверки корректного ввода количества строк.

3\. Внутри функции считать строку с помощью Console.ReadLine().

4\. Проверить корректность ввода с помощью Int32.TryParse.

5\. Если ввод корректный и число больше нуля, вернуть введённое значение.

6\. Если ввод некорректный, вывести сообщение об ошибке и повторить ввод.

7\. Определить функцию readLinesSequence, которая принимает количество строк count и возвращает последовательность seq<string>.

8\. Внутри readLinesSequence использовать цикл for _ in 1 .. count do и yield Console.ReadLine(), чтобы считать строки и добавить их в последовательность.

9\. Определить функцию countEven, которая принимает аккумулятор acc и строку line и возвращает acc + 1, если длина строки чётная, иначе возвращает acc.

10\. Определить функцию countEvenLength, которая принимает последовательность строк lines и с помощью Seq.fold countEven 0 lines возвращает количество строк с чётной длиной.

11\. Объявить точку входа программы [<EntryPoint>] let main argv =.

12\. Вывести пользователю сообщение "Введите количество строк:".

13\. Считать количество строк с помощью функции проверки ввода и сохранить результат в count.

14\. Вывести пользователю сообщение "Введите строки:".

15\. Вызвать readLinesSequence count и сохранить результат в lines типа seq<string>.

16\. Вызвать countEvenLength lines и сохранить результат в result.

17\. Вывести сообщение "Количество строк с чётной длиной: %d" и подставить result.

18\. Завершить программу, возвращая 0.

# Задание 3

## Задача 11

### Текст задачи

Выяснить, есть ли в каталоге файл с введенным названием.

### Алгоритм решения

1\. Подключить пространство имён System для работы с консолью.

2\. Подключить пространство имён System.IO для работы с файлами и каталогами.

3\. Определить функцию проверки корректного ввода пути к каталогу.

4\. Внутри функции считать путь с помощью Console.ReadLine().

5\. Проверить существование каталога с помощью Directory.Exists.

6\. Если каталог существует, вернуть введённый путь.

7\. Если каталог не существует, вывести сообщение об ошибке и повторить ввод.

8\. Определить функцию проверки имени файла.

9\. Внутри функции считать строку и проверить, что она не пустая.

10\. Если имя файла введено, вернуть его.

11\. Если строка пустая, вывести сообщение об ошибке и повторить ввод.

12\. Определить функцию fileExistsInDirectory, которая принимает путь к каталогу dirPath и имя файла fileName.

13\. Внутри функции вызвать Directory.EnumerateFiles(dirPath), чтобы получить последовательность всех файлов в каталоге.

14\. Применить Seq.exists с функцией (fun path -> Path.GetFileName(path) = fileName), чтобы проверить, есть ли файл с нужным именем.

15\. Объявить точку входа программы [<EntryPoint>] let main argv =.

16\. Вывести сообщение пользователю "Введите путь к каталогу:".

17\. Считать путь к каталогу с помощью функции проверки ввода и сохранить результат в dirPath.

18\. Вывести сообщение пользователю "Введите имя файла для поиска:".

19\. Считать имя файла с помощью функции проверки ввода и сохранить результат в fileName.

20\. Вызвать функцию fileExistsInDirectory dirPath fileName.

21\. Если функция вернула true, вывести "Файл "<fileName>" найден.".

22\. Если функция вернула false, вывести "Файл "<fileName>" не найден.".

23\. Завершить работу программы, возвращая 0.
