# Задание 2

## Задача 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для

последовательности

### Алгоритм решения

1\. Подключить пространство имён System для работы с консолью.

2\. Определить функцию readLinesSequence, которая принимает количество строк count и возвращает последовательность seq<string>.

3\. Внутри readLinesSequence использовать цикл for \_ in 1 .. count do и yield Console.ReadLine(), чтобы считать строки и добавить их в последовательность.

4\. Определить функцию countEven, которая принимает аккумулятор acc и строку line и возвращает acc + 1, если длина строки чётная, иначе возвращает acc.

5\. Определить функцию countEvenLength, которая принимает последовательность строк lines и с помощью Seq.fold countEven 0 lines возвращает количество строк с чётной длиной.

6\. Объявить точку входа программы \[<EntryPoint>] let main argv =.

7\. Вывести пользователю сообщение "Введите количество строк:".

8\. Считать количество строк с консоли и преобразовать его в int, сохранить в переменную count.

9\. Вывести пользователю сообщение "Введите строки:".

10\. Вызвать readLinesSequence count и сохранить результат в переменную lines типа seq<string>.

11\. Вызвать countEvenLength lines и сохранить результат в переменную result.

12\. Вывести сообщение "Количество строк с чётной длиной: %d" и подставить result.

13\. Завершить программу, возвращая 0.

# Задание 3

## Задача 11

### Текст задачи

Выяснить, есть ли в каталоге файл с введенным названием.

### Алгоритм решения

1\. Подключить пространство имён System для работы с консолью.

2\. Подключить пространство имён System.IO для работы с файлами и каталогами.

3\. Определить функцию fileExistsInDirectory, которая принимает путь к каталогу dirPath и имя файла fileName.

4\. Внутри функции вызвать Directory.EnumerateFiles(dirPath), чтобы получить последовательность всех файлов в каталоге.

5\. Применить Seq.exists с функцией (fun path -> Path.GetFileName(path) = fileName), чтобы проверить, есть ли файл с нужным именем.

6\. Объявить точку входа программы \[<EntryPoint>] let main argv =.

7\. Вывести сообщение пользователю "Введите путь к каталогу:".

8\. Считать путь к каталогу с консоли и сохранить в переменную dirPath.

9\. Вывести сообщение пользователю "Введите имя файла для поиска:".

10\. Считать имя файла с консоли и сохранить в переменную fileName.

11\. Вызвать функцию fileExistsInDirectory dirPath fileName.

12\. Если функция вернула true, вывести "Файл \\"<fileName>\\" найден.".

13\. Если функция вернула false, вывести "Файл \\"<fileName>\\" не найден.".

14\. Завершить работу программы, возвращая 0.
